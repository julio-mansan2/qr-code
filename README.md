### O que aprendi

<br>

- Importar fontes do Google 

````css

@import url('url da fonte desejada no google fonts');

````

<br>

- Centralizar elementos no centro da página com position absolute

````css

.div {
    
    position: absolute;
    margin: auto;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;

}

````
<br>

- Utilizar o border radius

````css

.div {
    border-radius: valor desejado;
}

````
